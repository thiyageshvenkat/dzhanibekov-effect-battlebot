# dzhanibekov-effect-battlebot

[Watch the demo video](flip_recording.mp4)

Simulates a battlebot abusing the Dzhanibekov effect or Tennis Racket effect. It occurs in the realistic simulations produced by `main.py`, but more clearly if starting high in the air, etc. I quickly made it to test the feasability of a battlebot that would flip between two axes for stylistic effect.

Uses pybullet for simulation.

Fake moments of inertia are used instead of properties like materials. Otherwise, generally accurate.
